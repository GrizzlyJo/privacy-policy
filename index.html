FRANÇAIS

1. Collecte de données

Notre bot affilié Amazon ne collecte aucune information personnelle auprès des utilisateurs. Les données utilisées proviennent uniquement de l’API Amazon et sont destinées à l’affichage automatisé d’offres promotionnelles.

2. Utilisation des APIs tierces

Nous utilisons les APIs suivantes :

  • Amazon Product Advertising API : pour obtenir des offres en temps réel
  • Facebook Graph API : pour publier sur votre page Facebook
  • Pinterest API : pour publier automatiquement sur votre tableau Pinterest
  • Discord Webhook : pour envoyer des notifications d’offres

Aucune donnée personnelle n’est extraite, stockée ou partagée.

3. Sécurité

Les clés API et jetons d’accès sont stockés de manière sécurisée via des variables d’environnement et ne sont jamais exposés publiquement.

4. Liens d’affiliation

Les liens publiés via ce bot sont des liens affiliés Amazon. Nous recevons une commission sur les ventes admissibles.

5. Modification

Nous nous réservons le droit de modifier cette politique à tout moment. Toute mise à jour sera publiée ici.

6. Contact

Pour toute question, contactez-nous via Discord ou votre méthode préférée.
